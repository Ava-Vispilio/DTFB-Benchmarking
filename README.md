# DTFB-Benchmarking (DFTB+ build & performance benchmarking)

This repo provides a reproducible workflow to **build DFTB+ from scratch** on **NSCC ASPIRE2A (PBS Pro)**, run a fixed benchmark suite, and collect performance results across:

- cores (single-node)
- nodes (MPI/OpenMP)
- compilers/flags
- libraries
- GPU on/off

Start here: [`docs/HowToRun.md`](docs/HowToRun.md)
